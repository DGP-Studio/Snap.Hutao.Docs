    Snap Hutao


    Toolkit Genshin Impact Multifungsi Open-Source 🧰

    MulaiDokumentasiPengenalan Proyek

    Desain Program yang Aman

    • Kendali Penuh atas Data Anda

      Data disimpan secara lokal secara default, tanpa bergantung pada layanan cloud atau platform pihak ketiga, memastikan otonomi Anda dalam mengakses data.

    • Perlindungan Data di Cloud

      Mendukung pencadangan ke server Snap Hutao Cloud khusus, memastikan data dapat dipulihkan dengan aman jika perangkat hilang.

    • Pengembangan Berbasis Komunitas

      Proyek ini bersifat open-source di GitHub di bawah lisensi MIT, menyambut tinjauan kode dan kontribusi dari komunitas. Pengemasan dan publikasi sepenuhnya otomatis melalui pipeline, tanpa intervensi manual, memastikan keamanan dan konsistensi.

    Kotak Alat Serba Guna

    Kuasai fitur-fitur inti dengan cepat dan jelajahi lebih banyak alat yang berguna.

    Aplikasi MSIX Windows Asli

    Menggunakan teknologi pengemasan aplikasi Windows asli, mendukung isolasi sandbox, memastikan stabilitas dan kebersihan sistem.

    Dasbor

    Menampilkan riwayat gacha, pencapaian, kalender game, dan pengumuman acara dengan cara paling sederhana, serta menyediakan akses cepat untuk meluncurkan game.

    Catatan Real-time

    Bebas dari batasan seluler, terima notifikasi catatan real-time di PC, terintegrasi dengan fitur notifikasi asli Windows.

    Catatan Spiral Abyss

    Simpan catatan Spiral Abyss pemain secara lokal, mendukung kueri catatan historis dan analisis statistik.

    Pergantian Multi-Akun

    Mendukung manajemen independen beberapa akun MiHoYo BBS, dengan mudah beralih dan menyimpan profil untuk setiap akun.

    Riwayat Gacha

    Menyediakan pergantian multi-UID, mengambil riwayat gacha dari file log atau data ekspor lainnya, dan menyimpannya secara permanen.

    Pencapaian

    Mendukung ekspor dan statistik data pencapaian, menyediakan manajemen target pencapaian tersembunyi.

    Peluncur Canggih

    Menyediakan fitur-fitur canggih seperti pergantian akun, pergantian server, dan pengaturan jendela, mengoptimalkan pengalaman peluncuran game.

    Database Hutao

    Menyediakan kueri dan berbagi komposisi tim komunitas, kombinasi senjata dan artefak.

    Teater Imaginarium

    Arsipkan catatan tantangan, kueri data historis tentang tingkat penggunaan setiap karakter.

    Statistik Karakter Saya

    Kueri atribut karakter spesifik, termasuk data detail tentang level, konstelasi, senjata, dan bakat.

    Paspor Snap Hutao

    Menyediakan pencadangan dan sinkronisasi data gacha di cloud, mendukung migrasi data di berbagai perangkat.

    Kalkulator Progres Peningkatan

    Buat rencana pengembangan karakter yang dipersonalisasi, mendukung banyak akun dan pencatatan item.

    Data Karakter

    Menyediakan informasi detail tentang cerita latar belakang karakter, materi pelatihan, dan rekomendasi pemain.

    Data Senjata

    Kueri atribut senjata dan cerita latar belakang, dan dapatkan rekomendasi pencocokan karakter.

    Data Monster

    Kueri atribut dan pengaturan monster, persiapkan diri untuk pertempuran di dalam game.

    Berkontribusi

    Snap Hutao adalah proyek gratis yang dikelola oleh komunitas open-source, dan kami menyambut lebih banyak pengembang untuk bergabung dengan kami untuk meningkatkan fungsionalitas dan kualitas Snap Hutao.

    1. Kirim Pull Request untuk meningkatkan kode dan fungsionalitas.
