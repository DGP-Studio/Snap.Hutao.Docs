    Desain Program yang Aman

    • Kendali Penuh atas Data

      Data disimpan secara default di perangkat lokal, tanpa bergantung pada layanan cloud atau platform pihak ketiga, memastikan otonomi dalam akses data.

    • Perlindungan Data di Cloud

      Mendukung pencadangan data ke Snap Hutao Cloud untuk server eksklusif, memastikan data tetap aman jika perangkat hilang.

    • Pengembangan yang Didorong oleh Komunitas

      Proyek ini bersifat sumber terbuka di GitHub dengan lisensi MIT, menyambut peninjauan dan kontribusi kode dari komunitas. Proses pengemasan dan perilisan sepenuhnya diselesaikan melalui alur otomatis tanpa campur tangan manual, memastikan keamanan dan konsistensi.

    Toolbox All-in-One

    Kuasai fitur-fitur inti dengan cepat, jelajahi lebih banyak alat yang berguna.

    Aplikasi MSIX Native Windows

    Mengadopsi teknologi pengemasan aplikasi native Windows, mendukung isolasi sandbox, memastikan stabilitas dan kebersihan sistem.

    Dasbor Data

    Menampilkan catatan Wish, progres pencapaian, kalender game, pengumuman acara dengan cara yang paling ringkas dan menyediakan akses cepat untuk memulai game.

    Catatan Real-time

    Mendobrak batasan perangkat seluler, menerima notifikasi Catatan miHoYo BBS secara real-time di PC, terintegrasi dengan fungsi notifikasi native Windows.

    Catatan Spiral Abyss

    Menyimpan catatan Spiral Abyss pemain secara lokal, mendukung kueri riwayat dan analisis statistik.

    Peralihan Multi-Akun

    Mendukung pengelolaan independen beberapa akun miHoYo BBS, beralih dengan mudah dan menyimpan profil setiap akun.

    Catatan Wish

    Menyediakan fungsi peralihan multi-UID, mendapatkan catatan Wish dari file log atau data ekspor lainnya, disimpan secara permanen.

    Catatan Pencapaian

    Mendukung ekspor dan statistik data pencapaian, menyediakan fungsi manajemen target pencapaian tersembunyi.

    Peluncur Tingkat Lanjut

    Menyediakan fungsi tingkat lanjut seperti peralihan akun, peralihan server, pengaturan jendela, dan lain-lain, mengoptimalkan pengalaman memulai game.

    Basis Data Hutao

    Menyediakan fungsi kueri dan berbagi data susunan tim, senjata, dan artefak yang direkomendasikan oleh komunitas pemain.

    Teater Fantasi

    Mengarsipkan catatan tantangan, menanyakan data riwayat tingkat penggunaan setiap karakter.

    Statistik Atribut

    Menanyakan atribut detail karakter, termasuk level, konstelasi, senjata, dan detail bakat.

    Paspor Hutao

    Menyediakan fungsi pencadangan dan sinkronisasi data Wish di cloud, mendukung migrasi data antar perangkat.

    Kalkulator Pengembangan

    Membuat rencana pengembangan karakter yang dipersonalisasi, mendukung multi-akun dan fungsi catatan item.

    Info Karakter

    Menyediakan informasi detail seperti latar belakang cerita karakter, material pelatihan, dan rekomendasi susunan tim pemain.

    Info Senjata

    Menanyakan atribut dan latar belakang cerita senjata, serta mendapatkan data rekomendasi susunan tim karakter.

    Info Monster

    Menanyakan atribut dan pengaturan monster, mempersiapkan pertempuran dalam game.

    Kontribusi

    Snap Hutao adalah proyek gratis yang dikelola oleh komunitas sumber terbuka, dan kami menyambut lebih banyak pengembang untuk bergabung, bersama-sama meningkatkan fungsi dan kualitas Snap Hutao.

    Berikut adalah cara Anda dapat berpartisipasi dalam kontribusi:

    1. Mengajukan Pull Request, untuk menyempurnakan kode dan fungsi.

    3. Menyempurnakan dokumentasi proyek, membantu lebih banyak pengguna untuk memahami dan menggunakan.
