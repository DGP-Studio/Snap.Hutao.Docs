    胡桃工具箱


    实用的开源多功能原神工具箱 🧰

    开始使用文档项目介绍

    安全的程序设计

    • 数据完全掌控

      数据默认存储在本地设备中，无需依赖云服务或第三方平台，保障数据访问的自主性。

    • 数据云端保护

      支持通过胡桃云备份至专属服务器，确保设备丢失时数据也能安全找回。

    • 社区驱动开发

      项目基于 MIT 协议在 GitHub 开源，欢迎社区代码审查与贡献。打包与发布全流程通过自动化流水线完成，无人工干预，确保安全性与一致性。

    多合一的工具箱

    快速掌握核心功能，探索更多实用工具。

    Windows 原生 MSIX 应用

    采用 Windows 原生应用打包技术，支持沙盒隔离，确保系统稳定性与整洁性。

    数据仪表盘

    以最简洁的方式展示祈愿记录、成就进度、游戏日历、活动公告并提供快速启动游戏的入口。

    实时便笺

    打破移动端限制，在 PC 上实时接收米游社便笺通知，集成 Windows 原生通知功能。

    深渊记录

    本地化保存玩家的深境螺旋记录，支持历史记录查询与统计分析。

    多账号切换

    支持多个米游社账号独立管理，轻松切换并保存各账号档案。

    祈愿记录

    提供多 UID 切换功能，从日志文件或其他导出数据中获取祈愿记录，永久保存。

    成就记录

    支持成就数据导出与统计，提供隐藏成就目标管理功能

    高级启动器

    提供账号切换、服务器切换、窗口设置等高级功能，优化游戏启动体验。

    胡桃数据库

    提供玩家社区配队、武器与圣遗物搭配数据查询与分享功能。

    幻想真境剧诗

    归档挑战记录，查询各角色出场率历史数据

    属性统计

    查询角色具体属性，包括等级、命之座、武器与天赋等详细数据。

    胡桃通行证

    提供祈愿数据云端备份与同步功能，支持多设备间数据迁移。

    养成计算器

    制定个性化角色养成计划，支持多账号与物品记录功能。

    角色资料

    提供角色背景故事、培养材料及玩家搭配推荐等详细信息。

    武器资料

    查询武器属性与背景故事，并获取角色搭配推荐数据。

    怪物资料

    查询怪物属性与设定，为游戏内战斗做好准备。
