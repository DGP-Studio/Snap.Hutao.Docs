# 🎉 已成功更新至 1.13.6 版本

**简体中文** | [English](/en/statements/latest.html)

- **【🎉新增】** 养成计划新增时区设置
  - 时区将在创建计划时设置
  - 创建计划后时区将无法修改
  - 现有养成计划将默认设置为胡桃设置中日历服务器对应时区
- **【🔨修复】** 修复了由其他应用启动导入成就不会显示导入弹窗的问题
- **【🔨修复】** 修复了实时便笺后台自动刷新时崩溃的问题 [#2491]
- **【✨优化】** 调整了遥测的方式，现在使用 Sentry 来更好的收集与处理异常信息
- **【✨优化】** 首次登录用户面板后，现在会优先选中米游社内配置的默认角色 [#2391]
- **【✨优化】** 修改了紧凑网页窗口的半透明逻辑
  - 现在窗口会在具有焦点时呈现不透明效果
  - 失去焦点时呈现半透明效果
