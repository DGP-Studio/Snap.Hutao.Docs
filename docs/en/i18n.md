# Localization

Snap Hutao primarily focuses on supporting the Genshin Impact Chinese server. Currently, we are gradually improving support for oversea servers, encompassing language localization and feature internationalization.
Due to the different types of restrictions on both types of server, the implementation of some features may vary. For example, the wish history export for oversea servers does not support the more convenient SToken refresh method.

## Participate in Globalization

- We have created an issue named `[Hoyolab]: Globalization support` on GitHub [Issue](https://github.com/DGP-Studio/Snap.Hutao/issues/638)
  - We will update the current internationalization progress of the main program in this issue.
  - You can announce your participation in the internationalization support of a certain part of the main program here, and we will provide support for it.
    - We also welcome direct Pull Requests through branches.
- **You can participate in the client translation work at [translate.hut.ao](https://translate.hut.ao)**
