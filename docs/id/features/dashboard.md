# Dasbor Beranda

![Dasbor](https://img.alicdn.com/imgextra/i2/1797064093/O1CN01Nlz8ca1g6e0tyxrBa_!!1797064093.png_.webp)

Saat Snap Hutao diluncurkan, secara default akan masuk ke dasbor beranda. Dasbor dibagi menjadi dua bagian: bagian atas menampilkan ikhtisar data permainan pribadi dalam bentuk kartu, dan bagian bawah menampilkan pengumuman permainan terbaru dan hitungan mundur acara. Kartu-kartu di halaman utama dapat dinonaktifkan atau diaktifkan kembali di pengaturan.

## Ikhtisar Fitur

- **Pengumuman Snap Hutao**

  - Jika tim pengembang merilis pengumuman baru, itu akan ditampilkan di bagian atas dasbor, dan sebagian besar pengumuman dapat ditutup.

- **Peluncur Game**

  - Pengguna dapat langsung memilih akun dan meluncurkan game dari dasbor.
    - Jika opsi "Gunakan login MiYouShe / HoYoLAB" diaktifkan, fitur pemilihan akun tidak akan tersedia.

- **Riwayat Permohonan**

  - Menampilkan UID arsip permohonan saat ini, dan menampilkan hitungan pity dalam bentuk bilah progres.
  - Untuk pengguna dengan beberapa arsip permohonan, arsip dapat dialihkan melalui tombol panah di kedua sisi area riwayat permohonan.

- **Statistik Pencapaian**

  - Menampilkan total pencapaian yang diselesaikan, persentase penyelesaian, dan pencapaian terbaru yang diperoleh dari arsip pencapaian.
  - Untuk pengguna dengan beberapa arsip pencapaian, arsip dapat dialihkan melalui tombol panah di kedua sisi area statistik pencapaian.

- **Catatan Real-time**

  - Menyediakan akses cepat ke data catatan real-time.
  - Untuk pengguna dengan beberapa catatan real-time, arsip dapat dialihkan melalui tombol panah di kedua sisi area catatan real-time.

- **Fitur Kalender**

  - Menampilkan karakter yang berulang tahun pada hari itu dan material Talenta, Karakter, dan Senjata yang dapat di-farming pada saat ini.
    - Jika "Rencana Pengembangan" berisi material yang dapat di-farming pada hari itu, ikon yang relevan akan disorot.
  - Data kalender didasarkan pada waktu komputer saat ini, dan pengguna dapat mengubahnya ke data zona waktu server tertentu di pengaturan.

- **Pengumuman Game**
  - Menyediakan penjelajahan cepat pengumuman game real-time.
  - Sumber data default adalah pengumuman resmi server domestik (国服), dan pengguna dapat beralih ke sumber data lain di pengaturan.

Dasbor dirancang untuk membantu pemain mengelola semua informasi penting yang terkait dengan Genshin Impact secara efisien, menyediakan akses mudah ke fungsi-fungsi dan tampilan data yang kaya.
